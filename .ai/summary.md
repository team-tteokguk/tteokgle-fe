## 🤖 AI Change Summary for PR #23

### Chunk 1/1
### 개요
이번 변경 사항은 사용자 프로필 관리 기능을 위한 새로운 컴포넌트와 스타일을 추가하는 것을 포함합니다. `InputWithLabel` 컴포넌트가 새로 추가되었으며, 사용자 마이페이지를 구성하는 `Mypage` 컴포넌트도 새로 생성되었습니다. 또한, CSS 파일에 몇 가지 스타일 수정이 이루어졌습니다.

### 주요 변경 사항
1. **InputWithLabel 컴포넌트 추가**: 사용자 입력을 위한 라벨과 아이콘을 포함하는 입력 필드 컴포넌트가 추가되었습니다.
2. **Mypage 컴포넌트 추가**: 사용자 정보를 관리할 수 있는 마이페이지 컴포넌트가 추가되었으며, 프로필 사진 업로드 및 사용자 이름, 상점 이름 입력 필드를 포함합니다.
3. **스타일 수정**: CSS 파일에서 입력 필드의 포커스 스타일을 리셋하고, 새로운 색상 및 그라데이션 변수를 추가했습니다.
4. **라우터 수정**: `Mypage` 컴포넌트가 라우터에 추가되어 사용자가 접근할 수 있도록 설정되었습니다.

### 위험/영향
- **UI/UX 변화**: 새로운 입력 컴포넌트와 마이페이지 레이아웃이 추가됨에 따라 사용자 경험이 개선될 것으로 예상됩니다.
- **기능적 결함 가능성**: 새로 추가된 컴포넌트와 스타일이 기존 시스템과 충돌할 가능성이 있으므로, 충분한 테스트가 필요합니다.

### 테스트/검증
- **기능 테스트**: `InputWithLabel` 및 `Mypage` 컴포넌트의 기능을 검증하기 위한 단위 테스트 및 통합 테스트가 필요합니다.
- **UI 테스트**: 새로운 UI 요소가 의도한 대로 작동하는지 확인하기 위한 시각적 테스트가 필요합니다.

### 후속 조치
- **API 통합**: `InputWithLabel` 컴포넌트의 저장 버튼 클릭 시 호출할 API를 구현해야 합니다.
- **모달 기능 개발**: 마이페이지에서 회원 탈퇴 버튼 클릭 시 모달을 통해 확인 절차를 추가해야 합니다.
- **테스트 작성**: 추가된 컴포넌트에 대한 테스트 케이스를 작성하여 코드의 안정성을 높여야 합니다.
