## 개요
이번 변경 사항은 방명록 API의 경로 수정과 인증 관련 API 및 훅을 추가하여 사용자 인증 및 방명록 기능을 개선한 내용입니다.

## 주요 변경 사항
1. **방명록 API 경로 수정**:
   - `useGuestBook.ts` 파일에서 `guestbookApi`의 경로를 `guestBookApi`로 변경하였습니다.

2. **인증 API 추가**:
   - `authApi.ts` 파일에 로그아웃 및 리프레시 토큰 기능을 포함한 `authApi` 객체가 추가되었습니다.

3. **인증 훅 추가**:
   - `useAuth.ts` 파일에 사용자 인증 상태를 관리하고 로그아웃 기능을 제공하는 `useAuth` 훅이 추가되었습니다.

4. **타입 정의 추가**:
   - `TokenResponse` 인터페이스가 `types/index.ts` 파일에 추가되어 인증 관련 데이터 구조를 정의하였습니다.

5. **Axios 인스턴스 추가**:
   - `axiosInstance.ts` 파일에 Axios 인스턴스를 생성하고, 요청 및 응답 인터셉터를 설정하여 인증 토큰을 관리하는 기능이 추가되었습니다.

6. **상태 관리 수정**:
   - `useAuthStore.ts` 파일에서 인증 상태를 관리하는 상태 구조가 수정되어 액세스 토큰을 포함하도록 변경되었습니다.

## 위험/영향
- 방명록 API 경로의 대소문자 변경으로 인해 해당 API를 호출하는 부분에서 오류가 발생할 수 있습니다. 이 변경이 다른 파일이나 모듈에 미치는 영향을 검토해야 합니다.
- 새로운 인증 API와 훅이 추가됨에 따라 기존 코드와의 호환성 문제는 없지만, 새로운 기능에 대한 테스트가 필요합니다.
